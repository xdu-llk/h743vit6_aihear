## PCB 改版说明

| 改动项 | 说明 |
|--------|------|
| **PA2 → ESP RST** | STM32 控制 ESP 硬复位。PA2 上电即输出高，检测到 ESP 死机后拉低 10ms 再拉高。PCB 走线直连，无需额外电阻 |
| **预留 TTL 排针** | PCB 上预留 4pin 排针（PA9/PA10/3.3V/GND），直连 STM32，用于串口日志查看 |
| **ESP IO2** | 悬空即可，ESP-01S 模组内部有上拉 |
| **ESP RST** | 目前悬空，改版后接 PA2 |
